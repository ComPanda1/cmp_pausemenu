# CMP_PAUSEMENU

------------

## REQUERIMENTS

------------

- [es_extended](http://https://documentation.esx-framework.org/legacy/installation/ "es_extended")

## COMO USARLO
1.  DESCARGAR
2.  PONER EN RESOURCES
3. CFG `cmp_pausemenu `
